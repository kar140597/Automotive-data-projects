# Automotive-data-projects
Building an open-source F1 Telemetry Dashboard here's a progress update!

Over the past few weeks I've been building a driver telemetry comparison tool powered by fastF1 and Plotly Dash. You can load any session (practice, qualifying, or race), pick any two drivers, and overlay their Speed, Throttle, Brake, Gear, RPM, and DRS traces all distance-aligned so corner comparisons are accurate.

Here's VER vs HAM from the 2025 Las Vegas GP qualifying. You can see exactly where Hamilton is heavier on the brakes into the chicane sequence and where Verstappen's throttle application is more aggressive on exit a glimpse into just how much information is hidden inside a single lap.

What's coming next:

→ Year-over-year comparison same race, multiple seasons overlaid. See how much faster cars have become at a specific circuit, or how a driver's style has evolved.

→ Race results panel final classification, gaps, fastest laps, and pit stop data alongside the telemetry for full context in one view.

The project is open source — link in the comments. Would love feedback from anyone into motorsport data, data engineering, or data viz. What would you want to see in a tool like this? Drop a comment
